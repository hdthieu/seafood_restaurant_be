# SOP QUẢN LÝ – SỰ CỐ KHẨN CẤP

## 1. Cháy nổ
- Ngắt điện/gas.
- Sử dụng bình chữa cháy.
- Sơ tán khách → gọi 114.

## 2. Khách bị tai nạn
- Hỗ trợ sơ cứu cơ bản.
- Báo bảo vệ + xe cứu thương nếu nặng.

## 3. Mất điện
- Bật nguồn máy phát (nếu có).
- Ngưng nhận món phức tạp.

## 4. Hệ thống POS bị lỗi
- Chuyển sang ghi order giấy tạm.
- Báo IT.
